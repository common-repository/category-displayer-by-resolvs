WP Category Displayer allows the possibility to display WordPress categories and other taxonomies on pages or posts.

== Description ==
WP Category Displayer allows you to creatively display WordPress categories and other taxonomies using beautifully designed and responsive templates.

= HOW IT WORKS =
Using the WP Category Displayer builder you can create unlimited Category Displayers which can be displayed on any page or post using a generated shortcode.

While creating a shortcode, the first steps are to select the necessary categories or taxonomies to be displayed. For WordPress categories, you can choose to display categories only or subcategories only.

WP Category Displayer also enables the possibility to add category images and display these categories through our beautiful responsive templates.

== Installation ==
1. Upload wp-category-displayer to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.

Alternatively:

1. Login to your WordPress admin area
2. Search for WP Category Displayer
3. Click install WP Category Displayer
4. Activate through the Plugins menu in WordPress or when asked during installation

== Frequently Asked Questions ==

= How does WP Category Displayer works? =
Using a built-in custom builder, WP Category Displayer generates shortcodes for the categories and/or subcategories selected to display. The shortcode will then creatively display the categories/subcategories through a selected template on a page or post.

= How to create a shortcode? =
Inside WP Category Displayer menu, select the function Create shortcode and follow the setup by entering a Shortcode name, selecting the categories or subcategories and the template to be used.
